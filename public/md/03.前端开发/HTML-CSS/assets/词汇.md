### **五、兼容性与适配（50 词）**

25. VH (Viewport Height) - 视口高度单位（1vh = 视口高度的 1%）
26. VMin (Viewport Minimum) - 视口最小单位（1vmin = min (vw, vh)）
27. VMax (Viewport Maximum) - 视口最大单位（1vmax = max (vw, vh)）
28. Rem Unit - Rem 单位（补充：root em，基于 html 元素 font-size，1rem = html 的 font-size）

34. Min () Function - Min 函数（min (a, b)，取较小值）
35. Max () Function - Max 函数（max (a, b)，取较大值）
36. Device Pixel Ratio (DPR) - 设备像素比（物理像素与逻辑像素的比值，如 Retina 屏 DPR=2）
37. Retina Display - Retina 显示屏（DPR≥2 的高清屏幕，需 2x 图片适配）

40. Media Query Breakpoint - 媒体查询断点（响应式设计的屏幕尺寸分界点，如 768px、1200px）

47. High Contrast Mode - 高对比度模式（系统高对比度模式，需适配的样式）
48. Reduced Motion - 减少动画模式（@media (prefers-reduced-motion: reduce)，适配系统减少动画设置）
