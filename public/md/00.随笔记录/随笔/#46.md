# #46

### 写前端

```markdown
这个登录页面代码可以分为模板结构（Template）、逻辑处理（Script）、样式设计（Style） 三个核心模块，每个模块又包含多个功能子模块，以下是详细讲解：
一、模板结构（Template）：页面骨架与交互入口
负责定义页面的 HTML 结构，包含所有可见元素和交互触发点，核心子模块如下：
1. 容器与面板基础结构
html
预览
<div class="login-body"> <!-- 页面主容器，承载背景图 -->
  <div 
    v-if="panelVisible" 
    class="login-panel draggable-panel"
    :style="{ left: panelPosition.left, top: panelPosition.top }"
    @mousedown="startDrag"
  > <!-- 登录面板核心容器 -->
功能：作为整个登录面板的载体，通过v-if="panelVisible"控制显示 / 隐藏，通过:style动态绑定位置（支持拖动），通过@mousedown触发拖动逻辑。
关键属性：v-if控制面板是否显示；:style绑定动态位置；@mousedown绑定拖动开始事件。
2. 面板头部（拖动与控制区）
html
预览
<div class="panel-header" @mousedown.stop>
  <h3 class="form-title">系统登录</h3>
  <div class="panel-controls">
    <span class="minimize-btn" @click="toggleMinimize">_</span>
    <span class="close-btn" @click="closePanel">×</span>
  </div>
</div>
功能：提供面板标题和控制按钮，支持最小化、关闭操作，且通过@mousedown.stop阻止头部点击事件冒泡（避免触发面板拖动）。
交互点：@click="toggleMinimize"触发最小化；@click="closePanel"触发关闭。
3. 登录表单（核心功能区）
包含用户输入、验证、按钮交互等核心登录功能，细分为：
输入项子模块：
用户名输入框：<el-input v-model="formData.account" />，通过v-model双向绑定表单数据。
密码输入框：type="password"隐藏输入内容，同样绑定formData.password。
验证码输入框：与验证码图片联动，支持输入和刷新。
操作项子模块：
记住我与忘记密码：<el-checkbox>绑定formData.rememberMe，链接提供跳转入口。
按钮组：包含 “账号登录” 和 “游客登录” 两个按钮，分别绑定login和guestLogin方法，通过:loading显示加载状态。
注册链接：提供注册入口，绑定toRegister方法。
4. 动态显示控制
通过v-show="!panelMinimized"控制表单内容的显示 / 隐藏（配合最小化功能）。
通过:class="{ loading: captchaLoading }"动态添加loading类，实现验证码加载状态的视觉反馈。
二、逻辑处理（Script）：数据管理与功能实现
基于 Vue 的<script setup>语法，负责数据定义、事件处理、API 交互等核心逻辑，分为以下子模块：
1. 数据管理子模块
javascript
// 表单数据
const formData = reactive({
  account: '',
  password: '',
  captcha: '',
  rememberMe: false
})

// 状态管理
const loading = ref(false) // 账号登录加载状态
const guestLoading = ref(false) // 游客登录加载状态
const captchaLoading = ref(false) // 验证码加载状态
const panelVisible = ref(true) // 面板显示状态
const panelMinimized = ref(false) // 面板最小化状态
const panelPosition = ref({ left: 'calc(100% - 320px - 20px)', top: '20px' }) // 面板位置
数据类型：使用reactive定义复杂表单对象（formData），使用ref定义简单状态变量（如加载状态、位置等）。
核心作用：存储表单输入值、控制 UI 状态（显示 / 隐藏、加载中）、记录面板位置。
2. 生命周期与事件监听
javascript
onMounted(() => {
  document.addEventListener('mousemove', handleMouseMove)
  document.addEventListener('mouseup', handleMouseUp)
  refreshCaptcha() // 初始化验证码
})

onBeforeUnmount(() => {
  document.removeEventListener('mousemove', handleMouseMove)
  document.removeEventListener('mouseup', handleMouseUp)
})
功能：在组件挂载时绑定鼠标移动 / 释放事件（支持拖动），并初始化验证码；在组件卸载前移除事件监听，避免内存泄漏。
3. 拖动功能子模块
javascript
const startDrag = (e) => { /* 记录拖动起点和偏移量 */ }
const handleMouseMove = (e) => { /* 计算并更新面板位置 */ }
const handleMouseUp = () => { /* 结束拖动状态 */ }
流程：
startDrag：鼠标按下时记录当前面板位置和鼠标偏移量，标记isDragging为true。
handleMouseMove：鼠标移动时，根据偏移量计算新位置，通过panelPosition更新面板坐标，并限制在窗口内。
handleMouseUp：鼠标释放时，标记isDragging为false，结束拖动。
4. 验证码功能子模块
javascript
const refreshCaptcha = async () => {
  captchaLoading.value = true
  try {
    const response = await fetch('/api/captcha') // 调用后端验证码API
    const blob = await response.blob()
    captchaUrl.value = URL.createObjectURL(blob) // 显示验证码图片
  } catch (error) { /* 错误处理 */ }
  finally { captchaLoading.value = false }
}
功能：通过调用后端/api/captcha接口获取验证码图片，使用URL.createObjectURL生成图片链接，配合captchaLoading显示加载状态。
5. 登录功能子模块
包含两种登录方式的逻辑：
账号登录：
javascript
const login = async () => {
  // 前端简单验证（非空检查）
  if (!formData.account) { ElMessage.warning('请输入用户名'); return }
  // 调用后端登录API：/api/login
  const response = await fetch('/api/login', {
    method: 'POST',
    body: JSON.stringify({ username: formData.account, password: formData.password, captcha: formData.captcha })
  })
  // 根据响应处理成功/失败逻辑
}

先做前端非空验证，再调用后端接口，携带用户名、密码、验证码，根据返回结果显示成功 / 失败提示。
游客登录：
javascript
const guestLogin = async () => {
  const response = await fetch('/api/guest/login', { method: 'POST' })
  // 处理登录结果
}

无需用户输入，直接调用后端游客登录接口/api/guest/login，适合临时访问场景。
三、样式设计（Style）：视觉呈现与交互反馈
使用 SCSS 语法定义页面样式，负责美化 UI 和交互反馈，核心子模块如下：
1. 基础样式与布局
scss
.login-body {
  width: 100vw; height: 100vh;
  background-image: url('/login.png'); /* 背景图 */
  background-size: cover;
  position: relative; /* 使面板可绝对定位 */
}

.draggable-panel {
  position: absolute; width: 320px;
  background-color: rgba(255, 255, 255, 0.9); /* 半透明背景 */
  border-radius: 14px; box-shadow: 0 10px 30px rgba(0, 0, 0, 0.12); /* 圆角与阴影 */
}
功能：设置页面全屏背景，定义登录面板的基础尺寸、定位方式、透明度、圆角和阴影，奠定整体视觉风格。
2. 交互反馈样式
通过伪类和动态类实现交互反馈：
悬停效果：按钮、控制图标等添加:hover样式，如：
scss
.minimize-btn:hover { background-color: #e8a933; transform: scale(1.1); }

输入框焦点效果：
scss
.form-input:focus {
  border-color: #409eff;
  box-shadow: 0 0 0 3px rgba(64, 158, 255, 0.2); /* 焦点高亮 */
}

加载状态样式：验证码加载时添加半透明遮罩和加载图标：
scss
.captcha-loading {
  position: absolute;
  display: flex; justify-content: center; align-items: center;
  background-color: rgba(255, 255, 255, 0.7);
}

3. 响应式与动态样式
面板最小化时，通过v-show="!panelMinimized"隐藏表单内容，仅保留头部。
拖动时通过:style动态更新left和top，配合 CSS 的transition: all 0.3s实现平滑移动效果。
总结：模块间的协作关系
Template 定义 “是什么”：通过指令（v-if、v-model）和事件（@click、@mousedown）连接数据和逻辑。
Script 定义 “做什么”：管理数据状态（formData、panelPosition）和实现功能逻辑（拖动、登录、验证码）。
Style 定义 “长什么样”：通过类名关联模板元素，美化 UI 并提供交互反馈（悬停、加载等）。
三个模块通过 Vue 的响应式机制联动：例如用户输入触发formData变化，进而更新 UI；点击登录按钮触发login方法，调用 API 后通过ElMessage反馈结果，同时更新loading状态控制按钮加载动画。
```
