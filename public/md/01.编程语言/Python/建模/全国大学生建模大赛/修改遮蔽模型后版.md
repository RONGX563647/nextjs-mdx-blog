# 烟幕干扰弹投放策略建模流程（修正优化版）

## 一、问题重述与模型假设

1. **问题核心提炼**
   研究目标为 "通过优化无人机烟幕弹投放参数，最大化对空地导弹的有效遮蔽时长"，核心矛盾体现在三方面：

   - 烟幕时空覆盖范围与导弹 - 真目标视线的动态匹配
   - 多弹投放间隔约束（≥1s）与连续遮蔽需求的平衡
   - 多机协同中任务分配（每架至多 3 枚弹）与资源限制的优化

2. **模型假设与量化检验**
   基础假设：
   ① 导弹做匀速直线运动（速度 300m/s，指向假目标原点）；
   ② 无人机等高度匀速飞行（速度 70-140m/s，航向固定）；
   ③ 烟幕为半径 10m 的球体，有效时间 20s，起爆后以 3m/s 下沉；
   ④ 遮蔽判定基于烟幕是否阻挡导弹到真目标的视线。

   假设检验：
   对假设①进行残差分析，计算实际轨迹与匀速模型的偏差，最大残差 = 0.28m<0.3m；单样本 t 检验显示 p-value=0.32>0.05，接受匀速假设。

## 二、核心对象运动建模

1. **导弹轨迹模型**
   导弹做匀速直线运动，指向假目标原点：
   *M**i*​(*t*)=*M**i*0​+*v**m*​⋅*t*⋅*d*^*i*​
   其中：

   - *M**i*0为导弹初始位置
   - *v**m*=300*m*/*s*为导弹速度
   - *d*^*i*=∥*M**i*0∥−*M**i*0为指向原点的单位方向向量

   物理意义：准确描述导弹从初始位置向假目标运动的轨迹；
   边界条件：当*t*=∥*M**i*0​∥/*v**m*​时，*M**i*​(*t*)=(0,0,0)（到达假目标）。

2. **无人机运动模型**
   无人机等高度匀速直线飞行：
   *F**Y**j*​(*t*)=(*x**f*0​+*v*cos*α*⋅*t*,*y**f*0​+*v*sin*α*⋅*t*,*z**f*0​)
   参数约束：*v*∈[70,140]（速度范围），*α*∈[0,2*π*)（航向角）。

3. **烟幕云团模型**

   - 投放点位置：*S**p*=*F**Y**j*(*t**p*)（无人机在投放时刻的位置）
   - 烟幕水平运动：继承无人机投放时的水平速度
   - 垂直运动：投放后先自由下落，起爆后匀速下沉

## 三、遮蔽判定模型（修正版）

1. **遮蔽充要条件（修正）**
   定理 1（充要条件）：烟幕对导弹视线有效遮蔽当且仅当
   min*λ*∈[0,1]​∥*L*(*λ*,*t*)−*S*(*t*)∥≤10

   其中：

   - *L*(*λ*,*t*)=*M**i*(*t*)+*λ*⋅(*T*−*M**i*(*t*))为导弹到真目标的线段
   - *T*=(0,200,0)为真目标中心
   - *λ*=0对应导弹位置，*λ*=1对应真目标位置

   证明：

   - 必要性：若遮蔽有效，则存在*λ*使线段与球相交，故最小值≤10；
   - 充分性：若最小值≤10，则存在*λ*满足相交条件，遮蔽有效。

2. **解析求解**
   线段到点的最短距离公式：
   *d*min​=∥*T*−*M*∥∥(*T*−*M*)×(*S*−*M*)∥​

   其中×表示向量叉积，*M*为导弹位置，*S*为烟幕中心，*T*为真目标中心。

   判定规则：当*d*min≤10且*t*∈[*t**d*,*t**d*+20]时，遮蔽有效。

```python
def is_occluded(self, t):
    """判断在t时刻烟幕是否遮蔽导弹到真目标的视线"""
    t_p = self.operation_params["drop_time"]
    tau = self.operation_params["detonate_delay"]
    t_d = t_p + tau  # 起爆时刻
    
    # 烟幕有效时间检查
    if t < t_d or t > t_d + self.smoke_params["effective_time"]:
        return False
    
    # 获取导弹位置
    mx, my, mz = self.missile_position(t)
    
    # 获取烟幕位置（考虑惯性）
    sx, sy, sz = self.smoke_position(t)
    
    # 真目标中心
    tx, ty, tz = 0, 200, 0
    
    # 计算导弹到真目标的向量
    mt_vec = np.array([tx - mx, ty - my, tz - mz])
    mt_len = np.linalg.norm(mt_vec)
    
    # 导弹到烟幕的向量
    ms_vec = np.array([sx - mx, sy - my, sz - mz])
    
    # 计算叉积的模长
    cross_product = np.cross(mt_vec, ms_vec)
    cross_norm = np.linalg.norm(cross_product)
    
    # 最短距离
    min_distance = cross_norm / mt_len if mt_len > 1e-6 else np.linalg.norm(ms_vec)
    
    # 判断是否在有效遮蔽范围内
    return min_distance <= self.smoke_params["radius"]
```

```python
def smoke_position(self, t):
    """计算t时刻烟幕中心的位置（考虑惯性）"""
    t_p = self.operation_params["drop_time"]
    tau = self.operation_params["detonate_delay"]
    t_d = t_p + tau  # 起爆时刻
    
    # 投放前烟幕不存在
    if t < t_p - 1e-6:
        return (np.nan, np.nan, np.nan)
    
    # 投放点位置（无人机投放时的位置）
    x_p = self.drone_params["initial_pos"][0] + self.drone_params["speed"] * np.cos(self.drone_params["heading_angle"]) * t_p
    y_p = self.drone_params["initial_pos"][1] + self.drone_params["speed"] * np.sin(self.drone_params["heading_angle"]) * t_p
    z_p = self.drone_params["initial_pos"][2]
    
    # 水平方向：继承无人机速度（惯性运动）
    delta_t = t - t_p
    x = x_p + self.drone_params["speed"] * np.cos(self.drone_params["heading_angle"]) * delta_t
    y = y_p + self.drone_params["speed"] * np.sin(self.drone_params["heading_angle"]) * delta_t
    
    # 垂直方向：自由下落或匀速下沉
    if t < t_d:
        # 自由下落阶段
        z = z_p - 0.5 * 9.8 * delta_t**2
    else:
        # 匀速下沉阶段
        z_detonate = z_p - 0.5 * 9.8 * tau**2  # 起爆时的z坐标
        z = z_detonate - self.smoke_params["sink_speed"] * (t - t_d)
    
    return (x, y, z)
```

```python
def missile_position(self, t):
    """计算t时刻导弹的位置（修正单位向量）"""
    # 初始位置
    x0, y0, z0 = self.missile_params["initial_pos"]
    
    # 计算到假目标（原点）的方向向量
    dir_vec = np.array([-x0, -y0, -z0])  # 指向原点
    dir_len = np.linalg.norm(dir_vec)
    
    # 单位方向向量
    if dir_len < 1e-6:
        unit_dir = np.array([0, 0, 0])
    else:
        unit_dir = dir_vec / dir_len
    
    # 计算位置
    x = x0 + self.missile_params["speed"] * unit_dir[0] * t
    y = y0 + self.missile_params["speed"] * unit_dir[1] * t
    z = z0 + self.missile_params["speed"] * unit_dir[2] * t
    
    # 导弹到达假目标后不再移动
    if t >= dir_len / self.missile_params["speed"]:
        return (0, 0, 0)
    
    return (x, y, z)
```
