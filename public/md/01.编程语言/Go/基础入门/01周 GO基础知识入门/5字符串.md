- 比较规则

  ：

  - 从左到右逐个字符比较ASCII/Unicode码点值
  - 第一个不同字符的码点值决定整体大小关系

## Go 语言的 `strings` 包

提供了丰富的字符串操作函数，涵盖了字符串的查找、替换、分割、连接、转换等常见功能。以下是对 `strings` 包核心方法的详细讲解，包含代码示例和注释说明：

### 一、字符串查找相关方法

用于在字符串中查找子串的位置或判断是否包含子串。

#### 1. `Contains(s, substr string) bool`

判断字符串 `s` 是否包含子串 `substr`。

go

```go
package main

import (
	"fmt"
	"strings"
)

func main() {
	s := "hello world"
	sub := "world"
	// 判断 s 是否包含 sub
	fmt.Println(strings.Contains(s, sub)) // 输出: true
	fmt.Println(strings.Contains(s, "go")) // 输出: false
}
```

#### 2. `ContainsAny(s, chars string) bool`

判断字符串 `s` 是否包含 `chars` 中的任意一个字符。

go

```go
func main() {
	s := "hello"
	// 判断 s 是否包含 'a' 或 'b' 或 'e'
	fmt.Println(strings.ContainsAny(s, "abe")) // 输出: true（包含 'e'）
	fmt.Println(strings.ContainsAny(s, "xyz")) // 输出: false
}
```

#### 3. `Index(s, substr string) int`

返回子串 `substr` 在 `s` 中第一次出现的索引，若不存在则返回 `-1`。

go

```go
func main() {
	s := "hello world"
	// 查找 "lo" 在 s 中的首次出现位置
	fmt.Println(strings.Index(s, "lo")) // 输出: 3（索引从 0 开始）
	fmt.Println(strings.Index(s, "go")) // 输出: -1（不存在）
}
```

#### 4. `LastIndex(s, substr string) int`

返回子串 `substr` 在 `s` 中最后一次出现的索引，若不存在则返回 `-1`。

go

```go
func main() {
	s := "ababa"
	// 查找 "aba" 在 s 中最后一次出现的位置
	fmt.Println(strings.LastIndex(s, "aba")) // 输出: 2（"aba" 从索引 2 开始："aba"[2:5]）
}
```

### 二、字符串替换相关方法

用于替换字符串中的子串。

#### 1. `Replace(s, old, new string, n int) string`

将 `s` 中前 `n` 个 `old` 子串替换为 `new`，`n < 0` 表示替换所有。

go

```go
func main() {
	s := "hello hello hello"
	// 替换前 2 个 "hello" 为 "hi"
	fmt.Println(strings.Replace(s, "hello", "hi", 2)) // 输出: hi hi hello
	// 替换所有 "hello" 为 "hi"（n = -1）
	fmt.Println(strings.Replace(s, "hello", "hi", -1)) // 输出: hi hi hi
}
```

#### 2. `ReplaceAll(s, old, new string) string`

替换 `s` 中所有 `old` 子串为 `new`（等价于 `Replace(s, old, new, -1)`）。

go

```go
func main() {
	s := "go is good, go is great"
	// 替换所有 "go" 为 "Golang"
	fmt.Println(strings.ReplaceAll(s, "go", "Golang")) // 输出: Golang is good, Golang is great
}
```

### 三、字符串分割与连接

用于将字符串分割为切片或拼接切片为字符串。

#### 1. `Split(s, sep string) []string`

以 `sep` 为分隔符将 `s` 分割为切片，若 `sep` 为空则按字符分割。

go

```go
func main() {
	s := "a,b,c,d"
	// 以 "," 分割字符串
	fmt.Println(strings.Split(s, ",")) // 输出: [a b c d]

	// 特殊情况：sep 为空时，按单个字符分割
	fmt.Println(strings.Split("hello", "")) // 输出: [h e l l o]
}
```

#### 2. `SplitN(s, sep string, n int) []string`

以 `sep` 为分隔符分割 `s`，最多分割为 `n` 个元素（`n < 0` 表示不限制）。

go



```go
func main() {
	s := "a,b,c,d"
	// 最多分割为 2 个元素
	fmt.Println(strings.SplitN(s, ",", 2)) // 输出: [a b,c,d]
}
```

#### 3. `Join(a []string, sep string) string`

将字符串切片 `a` 以 `sep` 为分隔符拼接为一个字符串。

go

```go
func main() {
	parts := []string{"hello", "world", "go"}
	// 用 " " 拼接切片元素
	fmt.Println(strings.Join(parts, " ")) // 输出: hello world go
	// 用 "," 拼接
	fmt.Println(strings.Join(parts, ",")) // 输出: hello,world,go
}
```

### 四、字符串转换相关方法

用于字符串的大小写转换、修剪等。

#### 1. `ToUpper(s string) string` / `ToLower(s string) string`

将字符串转换为全大写 / 全小写。

go

```go
func main() {
	s := "Hello World"
	fmt.Println(strings.ToUpper(s)) // 输出: HELLO WORLD
	fmt.Println(strings.ToLower(s)) // 输出: hello world
}
```

#### 2. `Trim(s, cutset string) string`

移除 `s` 首尾所有包含在 `cutset` 中的字符。

go

```go
func main() {
	s := "  hello world!  "
	// 移除首尾的空格（cutset 为 " "）
	fmt.Println(strings.Trim(s, " ")) // 输出: hello world!

	// 移除首尾的 "h"、"d"、"!"
	s2 := "hdhello!d"
	fmt.Println(strings.Trim(s2, "hd!")) // 输出: ello
}
```

#### 3. `TrimSpace(s string) string`

移除 `s` 首尾的空白字符（空格、制表符 `\t`、换行符 `\n` 等）。

go

```go
func main() {
	s := "\t\n hello \tworld \n"
	// 移除首尾空白字符
	fmt.Println(strings.TrimSpace(s)) // 输出: hello 	world（中间的制表符保留）
}
```

### 五、前缀与后缀判断

#### 1. `HasPrefix(s, prefix string) bool`

判断 `s` 是否以 `prefix` 为前缀。

go

```go
func main() {
	s := "http://example.com"
	fmt.Println(strings.HasPrefix(s, "http")) // 输出: true
	fmt.Println(strings.HasPrefix(s, "https")) // 输出: false
}
```

#### 2. `HasSuffix(s, suffix string) bool`

判断 `s` 是否以 `suffix` 为后缀。

go

```go
func main() {
	s := "example.txt"
	fmt.Println(strings.HasSuffix(s, ".txt")) // 输出: true
	fmt.Println(strings.HasSuffix(s, ".md")) // 输出: false
}
```

### 六、其他实用方法

#### 1. `Repeat(s string, count int) string`

将 `s` 重复 `count` 次并返回新字符串（`count <= 0` 时返回空字符串）。

go

```go
func main() {
	fmt.Println(strings.Repeat("ab", 3)) // 输出: ababab
	fmt.Println(strings.Repeat("x", 0)) // 输出: （空字符串）
}
```

#### 2. `Count(s, substr string) int`

统计 `substr` 在 `s` 中出现的非重叠次数。

go

```go
func main() {
	s := "ababa"
	fmt.Println(strings.Count(s, "aba")) // 输出: 1（"aba" 只在索引 0 出现一次，索引 2 开始是 "aba" 但与前重叠）
	fmt.Println(strings.Count(s, "ab")) // 输出: 2（"ab" 在索引 0 和 2 各出现一次）
}
```

### 总结

`strings` 包是 Go 中处理字符串的核心工具，上述方法覆盖了大部分日常开发需求。关键注意点：

- 所有方法均返回新字符串，原字符串不会被修改（Go 字符串是不可变的）。
- 方法参数中的 `sep`、`cutset` 等若为空字符串，可能有特殊行为（如 `Split` 按字符分割）。
- 索引相关方法返回 `-1` 表示未找到，需注意判断避免数组越界。

实际开发中，可结合具体场景选择合适的方法，例如：处理 URL 时用 `HasPrefix` 判断协议，处理 CSV 时用 `Split` 分割字段，格式化输出时用 `ToUpper`/`ToLower` 统一大小写等。

### 一、八股文题（20 题）

#### 基础篇（1-8 题）

1. ### **问题**：Go 语言中字符串的定义是什么？它的底层数据结构由哪两部分组成？
   
   **答案**：
   字符串是不可变的字节序列，用于表示文本数据。
   底层结构包含：
   
   - 指向字节数组的指针（`ptr *byte`）
   - 长度（`len int`）
     本质是`reflect.StringHeader`结构体。
2. **问题**：为什么说 Go 字符串是 "不可变的"？尝试修改字符串会发生什么？
   **答案**：
   - 不可变性：字符串的字节数组在创建后不能被修改，任何修改操作都会创建新的字符串。
   - 直接修改（如`s[0] = 'a'`）会导致编译错误；间接修改需先转换为`[]byte`或`[]rune`。
3. **问题**：如何在 Go 中创建字符串？有哪几种初始化方式？
   **答案**：
   - 字面量：`s := "hello"`
   - 转换：`s := string([]byte{104, 101, 108, 108, 111})`
   - 格式化：`s := fmt.Sprintf("name: %s", "Alice")`
   - 空字符串：`var s string`（零值为 ""，长度 0）
4. **问题**：`len("中")`和`len([]rune("中"))`的返回值分别是什么？为什么不同？
   **答案**：
   - `len("中")`返回 3："中" 的 UTF-8 编码占 3 字节。
   - `len([]rune("中"))`返回 1：`rune`按 Unicode **码点**计数，"中" 是一个码点。
     区别源于计数单位：字节 vs Unicode 字符。
5. **问题**：如何获取字符串的某个字符？直接使用索引`s[i]`得到的是什么？
   **答案**：
   - 获取字符：转换为`[]rune`后索引（`r := []rune(s); r[i]`）或使用`range`遍历。
   - `s[i]`返回第 i 个字节（uint8 类型），对于多字节字符（如中文）可能得到无效字节。
6. **问题**：`string`和`[]byte`、`[]rune`之间如何转换？转换的性能影响是什么？
   **答案**：
   - 转换：
     `b := []byte(s)`（字符串→字节切片）
     `s := string(b)`（字节切片→字符串）
     `r := []rune(s)`（字符串→rune 切片）
   - 性能：均会创建新的内存副本（因字符串不可变），大数据转换有性能开销。
7. ### **问题**：`range`遍历字符串时的行为是什么？与普通`for`循环有何区别？
   
   **答案**：
   
   - `range`遍历：自动解码 UTF-8，每次迭代返回当前`rune`的起始字节索引和码点值。
   - 普通`for`循环：按字节遍历，需手动处理多字节字符（可能得到拆分的字节）。
8. **问题**：空字符串（`""`）和`nil`的区别是什么？字符串是否可以为`nil`？
   **答案**：
   - 空字符串：长度为 0 的有效字符串（`len("") == 0`），底层指针可能指向空数组。
   - `nil`：字符串是值类型，不能为`nil`（`nil`仅适用于引用类型）。

#### 中级篇（9-15 题）

1. **问题**：如何高效拼接字符串？`+`运算符、`fmt.Sprintf`和`strings.Builder`的性能差异是什么？
   **答案**：

   - 高效方式：`strings.Builder`（预分配内存，避免多次复制）。
   - 性能：`strings.Builder` > `bytes.Buffer` > `fmt.Sprintf` > `+`运算符。
   - 原因：`+`每次拼接创建新字符串，`strings.Builder`直接在内部缓冲区写入。

2. **问题**：字符串的比较（`==`）是如何进行的？比较规则是什么？
   **答案**：

   - 比较逻辑：先比较长度，长度不同则不等；长度相同则逐字节比较。
   - 规则：按字节序列完全一致才相等，与编码方式无关（如不同 UTF-8 序列表示同一字符则不等）。

   **答案**：

   - `strings.Contains(s, substr)`：判断是否包含。
   - `strings.Index(s, substr)`：返回子串首次出现的索引（-1 表示不包含）。
   - `strings.ContainsAny(s, chars)`：判断是否包含 chars 中的任一字符。

   **答案**：

1. ### **问题**：字符串的内存分配策略是什么？短字符串和长字符串的存储有何区别？

   - 高效方式：
     - 使用`strings.Builder`或`bytes.Buffer`拼接。
     - 避免频繁转换（`string` ↔ `[]byte`）。
     - 分片处理（避免一次性加载全部到内存）。
   - 注意：内存占用（字符串不可变导致的副本）、GC 压力、缓存友好性。

3. **问题**：什么是 "字符串 intern 机制"？Go 是否支持？有什么优缺点？
   **答案**：

   - 机制：将相同字符串存储为单份副本（共享底层数据），减少内存占用。
   - Go 支持：编译期对字符串字面量自动 intern，运行时需手动实现（如用`map`缓存）。

   ```go
       prev := rune(s[0])
       count := 1

       }
       return s
   }
   ```

   ```go
   func fuzzyMatch(s, pattern string) bool {
       // 递归终止条件
       if pattern == "" {
           return s == ""
       }
       if s == "" {
           // 剩余pattern必须全是*才匹配
           for _, r := range pattern {
               if r != '*' {
                   return false
               }
           }
           return true
       }
       
       // 处理*通配符
       if pattern[0] == '*' {
           // *匹配0个或多个字符
           return fuzzyMatch(s, pattern[1:]) || fuzzyMatch(s[1:], pattern)
       }
       
       // 处理?通配符或普通字符
       if pattern[0] == '?' || s[0] == pattern[0] {
           return fuzzyMatch(s[1:], pattern[1:])
       }
       
       return false
   }
   
   // 使用：
   // fmt.Println(fuzzyMatch("hello", "h*o"))   // true
   // fmt.Println(fuzzyMatch("hello", "h?llo")) // true
   // fmt.Println(fuzzyMatch("hello", "h?l?"))  // false
   ```
   
   说明：递归实现通配符匹配，`*`可匹配任意长度字符序列（包括空），`?`匹配单个字符。

   ```go
   func longestUniqueSubstring(s string) string {
       start := 0
       maxLen := 0
       maxStart := 0
       lastSeen := make(map[rune]int) // 记录字符最后出现的位置
       
       for i, r := range s {
           // 若字符已出现且在当前窗口内，移动窗口起始位置
           if pos, ok := lastSeen[r]; ok && pos >= start {
               start = pos + 1
           }
           // 更新最大长度
           if i-start+1 > maxLen {
               maxLen = i - start + 1
               maxStart = start
           }
           lastSeen[r] = i
       }
       // 返回最长子串
       return s[maxStart : maxStart+maxLen]
   }

   ```

   ```go
   func caesarCipher(s string, shift int) string {
       // 处理移位（确保在0-25之间）
       shift = shift % 26
       if shift < 0 {
           shift += 26
       }
       
       runes := []rune(s)
       for i, r := range runes {
           switch {
           case r >= 'a' && r <= 'z':
               // 小写字母
               runes[i] = (r-'a'+rune(shift))%26 + 'a'
           case r >= 'A' && r <= 'Z':
               // 大写字母
               runes[i] = (r-'A'+rune(shift))%26 + 'A'
           }
           // 非字母字符不处理
       }
       return string(runes)
   }
   ```

5. ### **场景**：实现字符串的 Levenshtein 距离（编辑距离）计算，即最少操作次数（插入、删除、替换）使两字符串相等。
   
   **答案**：
   
   go
   
   ```go
   func levenshteinDistance(s, t string) int {
       m, n := len(s), len(t)
       // 创建DP表
       dp := make([][]int, m+1)
       for i := range dp {
           dp[i] = make([]int, n+1)
       }
       
       // 初始化边界
       for i := 0; i <= m; i++ {
           dp[i][0] = i // s[0..i-1]到空字符串的距离
       }
       for j := 0; j <= n; j++ {
           dp[0][j] = j // 空字符串到t[0..j-1]的距离
       }
       
       // 填充DP表
       for i := 1; i <= m; i++ {
           for j := 1; j <= n; j++ {
               if s[i-1] == t[j-1] {
                   // 字符相同，距离不变
                   dp[i][j] = dp[i-1][j-1]
               } else {
                   // 取插入、删除、替换的最小值+1
                   dp[i][j] = min3(
                       dp[i-1][j]+1,   // 删除
                       dp[i][j-1]+1,   // 插入
                       dp[i-1][j-1]+1, // 替换
                   )
               }
           }
       }
       return dp[m][n]
   }
   
   func min3(a, b, c int) int {
       minVal := a
       if b < minVal {
           minVal = b
       }
       if c < minVal {
           minVal = c
       }
       return minVal
   }
   
   // 使用：
   // fmt.Println(levenshteinDistance("kitten", "sitting")) // 3（k→s, e→i, 加g）
   // fmt.Println(levenshteinDistance("abc", "adc"))       // 1（替换b→d）
   ```
   
   说明：动态规划经典应用，通过构建 DP 表计算两个字符串的编辑距离，可用于拼写检查、相似度比较等场景。
