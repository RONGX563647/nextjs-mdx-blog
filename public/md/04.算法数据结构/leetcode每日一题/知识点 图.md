## 1.图论

>  图论是研究由顶点和边构成的图结构的数学分支，用于描述对象间关系。图分无向/有向、加权/无权等类型，涉及度数、路径、连通性等概念。经典算法包括最短路径（Dijkstra等）、最小生成树（Kruskal等）。应用于计算机网络、社交分析、物流规划等领域，近年与机器学习结合形成图神经网络。

### 图论：用 “图” 解析复杂关系的数学工具

#### 一、图论的基本概念

**图论**是数学的一个分支，主要研究 “图” 的结构和性质。这里的 “图” 并非传统意义上的图像，而是由**顶点（节点）\**和\**边**组成的抽象结构，用于描述对象之间的关系。

- **顶点（Vertex）**：代表研究对象（如人、城市、计算机等），用圆圈表示。
- **边（Edge）**：代表对象之间的关系（如友谊、道路、网络连接等），用线段或曲线表示。

**示例**：

- 社交网络中，顶点是用户，边表示 “关注” 关系。
- 交通地图中，顶点是城市，边是城市间的道路。

#### 二、图的分类

根据边的特性，图可分为多种类型：

1. **无向图 vs 有向图**
   - **无向图**：边没有方向（如 “朋友关系” 是双向的），边用不带箭头的线段表示。
   - **有向图**：边有方向（如 “关注” 是单向的），边用带箭头的线段表示。
2. **加权图 vs 无权图**
   - **加权图**：边上带有数值（权重），表示关系的强度或成本（如道路的距离、网络带宽）。
   - **无权图**：边仅表示是否存在关系，无数值属性。
3. **简单图 vs 多重图**
   - **简单图**：任意两个顶点之间最多一条边，且无自环（顶点到自身的边）。
   - **多重图**：允许两个顶点之间有多条边，或存在自环。

#### 三、图论的关键术语与性质

1. **顶点的度数（Degree）**
   - 无向图中，顶点的度数是与之相连的边的数量。
   - 有向图中，度数分为**入度**（指向该顶点的边数）和**出度**（从该顶点出发的边数）。
2. **路径（Path）与回路（Cycle）**
   - **路径**：顶点序列通过边连接（如 A→B→C 是一条路径）。
   - **简单路径**：顶点不重复的路径。
   - **回路**：起点和终点相同的路径（如 A→B→A）。
3. **连通性（Connectivity）**
   - **连通图**：任意两个顶点之间存在路径。
   - **非连通图**：图中存在至少两个顶点无法通过路径相连，可分为多个**连通分量**。
4. **子图（Subgraph）**
   - 从原图中选取部分顶点和边组成的新图，保留顶点之间的关联关系。

#### 四、图论的经典问题与算法

图论在计算机科学、工程、社会科学等领域有广泛应用，以下是一些核心问题及解决方案：

1. **最短路径问题**

   - **问题**：在加权图中找到两点之间总权重最小的路径。

   - 算法

     ：

     - **Dijkstra 算法**：适用于无权图或非负权图，贪心策略求解。
     - **Bellman-Ford 算法**：可处理负权边，但时间复杂度较高。
     - **Floyd-Warshall 算法**：多源最短路径，动态规划思想。

2. **最小生成树（MST）问题**

   - **问题**：在连通无向加权图中，找到一个包含所有顶点的树结构，且边权之和最小。

   - 算法

     ：

     - **Kruskal 算法**：按边权排序，用并查集避免环，贪心构建。
     - **Prim 算法**：从一个顶点出发，逐步添加最小权边连接新顶点。

3. **拓扑排序**

   - **问题**：对有向无环图（DAG）的顶点排序，使得所有有向边从排在前面的顶点指向后面的顶点（如任务调度顺序）。
   - **算法**：基于深度优先搜索（DFS）或计算入度的贪心算法。

4. **欧拉路径与哈密顿路径**

   - **欧拉路径**：经过每条边恰好一次的路径，存在条件：图连通，且 0 或 2 个顶点度数为奇数。
   - **哈密顿路径**：经过每个顶点恰好一次的路径，NP 难问题，无高效通用算法。

### 通用结构

```java
interface Graph {
    // 添加一条边（带权重）
    void addEdge(int from, int to, int weight);

    // 删除一条边
    void removeEdge(int from, int to);

    // 判断两个节点是否相邻
    boolean hasEdge(int from, int to);

    // 返回一条边的权重
    int weight(int from, int to);

    // 返回某个节点的所有邻居节点和对应权重
    List<Edge> neighbors(int v);

    // 返回节点总数
    int size();
}
```
