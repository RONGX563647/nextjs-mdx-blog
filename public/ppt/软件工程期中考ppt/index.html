📤 Node.java 核心方法

public class Node { String nodeId; String host; int port; List<String> peerNodes; Blockchain blockchain; void addPeer(String addr) // 添加邻居 void removePeer(String addr) // 移除邻居 void broadcastBlock(Block b) // 广播区块 void broadcastTransaction(t) // 广播交易 void syncWithNetwork() // 链同步 List<Block> requestChainFromPeer(p) boolean isValidChain(chain) // 链验证 }

🔄 最长链原则(实际代码)

public void syncWithNetwork() { for (String peer : peerNodes) { List<Block> peerChain = requestChainFromPeer(peer); if (peerChain != null && peerChain.size() > 0) { if (isValidChain(peerChain) && peerChain.size() > blockchain.getChain().size()) { blockchain.setChain(peerChain); } } } }

📤 ChatProducer (6种消息)

public class ChatProducer { @Autowired RabbitTemplate rabbitTemplate; // Fanout: chat-fanout-exchange sendChatMessage(content, sender) sendNotificationMessage(c, s) // Direct: chat-direct-exchange sendInfoMessage(content, sender) // route.info sendWarningMessage(content, sender) // route.warning sendErrorMessage(content, sender) // route.error // Topic: chat-topic-exchange sendOrderMessage(content, sender) // order.create }

📥 ChatConsumer (6个监听)

@Component public class ChatConsumer { @RabbitListener(queues="chat-queue") public void receiveChatMessage(MessageDTO msg) @RabbitListener(queues="info-queue") public void receiveInfoMessage(MessageDTO msg) @RabbitListener(queues="warning-queue") public void receiveWarningMessage(MessageDTO msg) @RabbitListener(queues="error-queue") public void receiveErrorMessage(MessageDTO msg) @RabbitListener(queues="order-queue") public void receiveOrderMessage(MessageDTO msg) @RabbitListener(queues="notification-queue") public void receiveNotificationMessage(MessageDTO msg) }
RabbitMQConfig 配置：3种交换机(Direct=chat-direct-exchange/Topic=chat-topic-exchange/Fanout=chat-fanout-exchange) + 6个队列(chat-queue/info-queue/warning-queue/error-queue/order-queue/notification-queue) + Jackson2JsonMessageConverter JSON序列化 + 路由键(route.info/route.warning/route.error/order.create)

✨ 核心功能 & 总结

👤
用户模块
注册自动生成RSA钱包+专属节点ID，BCrypt密码加密，UserRepository JPA存储到H2
⛏️
PoW 挖矿
difficulty=4，打包交易生成新区块，挖矿奖励10.0，SHA256哈希计算
🔒
数据防篡改
SHA-256链式结构，isChainValid()验证每个区块哈希和前一区块哈希
🌐
P2P 网络
WebClient HTTP通信，broadcastBlock广播，syncWithNetwork最长链同步
💬
RabbitMQ聊天
3种交换机(Direct/Topic/Fanout)，6种消息类型，6个队列独立消费
🎯
ChatService
模拟用户登录/交易讨论场景，sendCustomMessage自定义消息类型
项目技术栈总结：Spring Boot 2.7 + JDK 17 + H2数据库 + WebFlux WebClient(P2P) + RabbitMQ(3种交换机/6种消息) + Vue3(前端) + BCrypt+SHA256+RSA(安全) + DevTools(热部署)
