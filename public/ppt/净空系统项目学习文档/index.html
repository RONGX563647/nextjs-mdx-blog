文档中心

风电净空监控系统技术文档与教程汇总

文档概览

本文档中心汇集了风电净空监控系统的业务页面说明和技术教程，涵盖前端页面开发、后端架构设计、数据库优化等多个方面。

系统基于 JeecgBoot 3.4.0 低代码平台构建，采用 Vue 2.6 + Ant Design Vue 前端技术栈，结合 Spring Boot 2.6.6 后端框架，提供完整的设备管理、视频监控、净空检测统计分析功能。

业务页面文档

数据总览页面

系统主数据大屏，展示关键指标和实时监控信息，包含摄像头监控、设备状态总览等功能。

数据分析报告页面

提供净空检测数据的深度分析功能，支持多种图表展示和数据对比分析。

风轮不平衡评估报告页面

针对风机风轮不平衡问题的专项评估报告，包含振动分析、平衡状态评估等。

技术教程文档

分库分表实战教程

详解如何在实际项目中进行数据库分库分表，提升系统性能和扩展性。

定时任务全链路教程

基于 Quartz 的定时任务完整实现方案，包含任务调度、监控和管理。

数据流

系统数据流转全景，从边缘设备到服务端的数据采集、处理和存储流程。

Redis 教程

Redis 在系统中的应用实践，包括缓存策略、分布式锁等场景。

字典使用深入讲解

JeecgBoot 字典功能的深度使用指南，包含字典配置、前端展示等。

快速开始

后端启动

cd jeecg-boot-module-system
mvn spring-boot:run

前端启动

cd fengji-system-vue
yarn serve
提示：后端已配置热部署（devtools），修改 Java 代码后无需重启服务。
